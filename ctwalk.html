Step 1 - Viewing the Song List

The Song List screen contains an alphabetical list of each of your music charts.



From this screen, you can create a new music chart, display an existing chart or create a play list.






Step 2 - Create a Play List


The Play List screen lists a selected range of music charts from the Song List screen that can be reordered and removed by using the play list screen.



The music charts can then be displayed sequentially from within the Song Detail screen.





Step 3 - Viewing the Song Detail

The Song Detail screen displays each music chart in detail.



Each music chart stores the artist, song name, tempo as well as eight hand written sections that can be used to store chords, notes or rhythms. The section labels can also be changed to correctly order the song.



The tempo can be set manually or by using tap tempo. WAV files can be played from the screen instead of the inbuilt metronome.
